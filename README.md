# 📊 SmartEDA – Data Intelligence Platform

## 🚀 Live Demo

Try SmartEDA here:

https://smarteda-data-intelligence-platform-hslfdvipfdykbmdzq8tyoo.streamlit.app

SmartEDA is an interactive data analysis and machine learning web application built using Python and Streamlit.

The platform allows users to upload CSV datasets and automatically perform Exploratory Data Analysis (EDA), data quality checks, visualization, data cleaning, machine learning, clustering, and report generation.

---

## 🚀 Features

### 📁 CSV Dataset Upload

- Upload CSV datasets
- Start analysis using the Start Analysis button
- View dataset information
- Try built-in sample datasets

### 📋 Dataset Overview

- Number of rows and columns
- Missing value percentage
- Duplicate row detection
- Data quality score
- Column statistics
- Raw data preview

### 🧠 Automated Insights

- Automatic data analysis
- Missing value detection
- Duplicate detection
- Data quality observations
- Rule-based insights
- Mini Query System

### 📊 Interactive Visualizations

- Histograms
- Box plots
- Violin plots
- Scatter plots
- Correlation heatmap
- Missing value visualization
- Categorical charts
- Interactive dashboard

### 🤖 Automated Machine Learning

- Automatic problem type detection
- Classification and Regression
- Random Forest models
- Model performance evaluation
- Feature importance analysis
- Confusion matrix
- Classification report
- RMSE calculation

### 🧬 K-Means Clustering

- Select numerical features
- Choose the number of clusters
- Perform automatic clustering
- Interactive cluster visualization
- Cluster summary

### 💡 Smart Recommendations

- Automatic data improvement suggestions
- High, medium, and low priority recommendations
- Machine learning preparation suggestions

### 🧹 Data Cleaning

- Remove missing values
- Fill missing values using mean, median, or mode
- Remove duplicate rows
- Remove outliers using the IQR method
- Reset to the original dataset
- Download the cleaned dataset

### 🔍 Smart Column Analyzer

- Analyze individual columns
- Detect numerical and categorical columns
- View missing and unique values
- Statistical analysis
- Distribution visualization
- Skewness analysis

### 📄 Reports and Export

- Generate PDF analysis reports
- Download cleaned CSV datasets
- Generate project resume bullet points
- View session history
- View top correlations

### 🎤 Voice Commands

SmartEDA includes browser-based voice navigation for accessing different analysis sections.

## ⚙️ Installation

### 1. Clone the repository

git clone YOUR_REPOSITORY_URL

### 2. Open the project folder

cd smarteda

### 3. Install the required libraries

pip install -r requirements.txt

### 4. Run the application

python -m streamlit run app.py

---

## 💻 How to Use

1. Open the SmartEDA application.
2. Login using the demo login system.
3. Upload a CSV dataset or select a sample dataset.
4. Click the Start Analysis button.
5. Explore the dataset overview and automated insights.
6. Create interactive visualizations.
7. Run machine learning models.
8. Perform K-Means clustering.
9. Clean the dataset.
10. Analyze individual columns.
11. Generate reports and download the cleaned dataset.
